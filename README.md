# Projeto Agenda de Contatos em Python

Este projeto foi desenvolvido como prática de programação em Python, com foco em lógica, manipulação de dados e persistência em arquivos CSV.

## Objetivo

Criar um sistema de agenda de contatos em modo terminal, permitindo:

- Adicionar contatos
- Buscar contatos
- Editar contatos
- Excluir contatos
- Exportar dados para CSV
- Importar dados de CSV
- Salvar automaticamente os registros

---

## Tecnologias utilizadas

- Python 3
- Manipulação de arquivos `.csv`
- Estruturas de dados (`dict`)
- Controle de fluxo (`if`, `elif`, `else`)
- Tratamento de exceções (`try`, `except`)
- Funções reutilizáveis

---

## Conceitos aplicados

Durante o desenvolvimento deste projeto, pratiquei:

### Variáveis e Tipos de Dados
Uso de strings, listas e dicionários para armazenar informações.

### Estruturas Condicionais
Aplicação de `if`, `elif` e `else` para controlar o fluxo do sistema.

### Funções
Criação de funções para organizar o código e reutilizar lógica.

### Dicionários
Armazenamento estruturado de contatos em memória.

### Manipulação de Arquivos
Leitura e escrita em arquivos CSV para persistência de dados.

### Tratamento de Erros
Uso de `try` e `except` para evitar falhas inesperadas.
